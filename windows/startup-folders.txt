## Windows NT 6.0 - 10.0 (Windows Vista / Windows Server 2008 to Windows 10 / Windows Server 2019)
# elevated
<systemDrive>:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

# userland 
<systemDrive>:\Users\<user>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

## Windows NT 5.0 - 5.2 (Windows 2000 - Windows XP / Windows Server 2003 R2)
# elevated 
<systemDrive>:\Documents and Settings\All Users\Start Menu\Programs\Startup

# userland 
<systemDrive>:\Documents and Settings\<user>\Start Menu\Programs\Startup
